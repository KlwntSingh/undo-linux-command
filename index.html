undo-linux-command

simple utility which will undo the linux command.

When you start terminal, First run alias.sh. and than you can undo any last command using zz command.

This utility currently supports only two commands i.e only two command can be undone and they are cp and mv command with sudo and with options but option should be with one -(hyphen sign) otherwise utility may break. Future supports may include tar command and rm command.
